대학교 자료구조 과목에서 정점이 100개, 간선이 100개인 희소 그래프와 간선이 4000개인 밀집 그래프에 대하여 각각 인접 행렬 방식과 인접 리스트 방식으로 그래프를 표현하고 희소그래프-인접행렬 / 희소그래프-인접리스트 / 밀집그래프-인접행렬 / 밀집그래프-인접리스트 4가지 경우에 대해 성능 측정을 진행해. 성능 측정은 그래프를 사용하는데 사용한 메모리 용량, 추가적으로 간선을 삽입/삭제할 때 비교 횟수, 두 정점의 연결 여부를 확인할 때 비교 횟수, 특정 노드에 연결된 인접 노드를 출력할 때 비교 횟수야.
밀집 그래프는 배열, 회소 그래프는 연결 자료형으로 구현해.
언어는 C언어로 구현해야 되며 출력 양식은 아래와 같아.

케이스 1:
메모리 X Bytes
간선 삽입/삭제 비교 N번
두 정점의 연결 확인 비교 M번
한 노드의 인접 노드 출력 비교 L번

위 형식 외에 다른 출력(인접 노드나 다른 글자 등)은 없어야 해.
프로그램 작성 후 각 케이스 별(4개)로 메모리 공간, 간선 삽입/삭제 비교, 두 정점의 연결 확인 비교, 한 노드의 인접 노드 출력 비교의 횟수가 평균에서 크게 벗어나지 않는지 확인하고, 만약 벗어날 경우 프로그램에서 오류가 있는지 확인해서 고쳐줘.
